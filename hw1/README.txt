How to use:
		Use the command inside the same folder:  
			go run q2.go  -pairs <int> -count <int> -v

		The -v flag is optional, -v makes the output verbose
		this allows to vizualize the concurrency.


Example:	go run q2.go -pairs 10 -count 100 
Result:		Will run 10 producers and consumers counting from to 100
